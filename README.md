# 🎲 Dice Roll Simulator

A simple Python GUI project where users guess a dice number, roll the dice, and view results with a graph.

## 🚀 Features

* Tkinter GUI
* Dice images
* Random dice roll
* Guess validation
* Result messages
* Graph of guesses vs actual values

## 🛠️ Tech Used

* Python
* Tkinter
* Pillow
* Matplotlib

## 📦 Install

```bash
pip install -r requirements.txt
```

## ▶️ Run

```bash
python dice_simulator.py
```
